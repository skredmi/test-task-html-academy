тестовое задание


Необходимо сверстать и оживить небольшой виджет новостной ленты

Требования:

1) Подключение виджета сделать через JavaScript, который вставляется на страницу
2) Из скрипта получается весь набор данных, из которых формируется лента новостей.
3) После загрузки данных в нижней части страницы появляется иконка/картинка с отображением количества сообщений в ленте. При клике на иконку открывается виджет со списком сообщений
4) Одно сообщение ленты содержит поля:

а) Заголовок
б) Автор
в) Дата и время
г) Ссылка на подробности
д) Статус (прочитано или нет)

Внешнее отображение виджета придумать самостоятельно или взять за основу пример какого-либо сервиса.
Дизайн оцениваться не будет.
Дополнительно можно добавить интерактивности и по какому-то действию отмечать сообщение прочитанным.
